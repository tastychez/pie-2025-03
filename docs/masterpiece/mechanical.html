Gantry & Motion System

The core of Masterpiece’s motion system is a belt-driven XY gantry mounted beneath the chessboard. Using linear rails, idlers, and belt tensioners, the gantry can position an end effector under any square with repeatable accuracy while avoiding racking or binding. Hard end stops and homing routines prevent over-travel and give us a consistent reference for all motion.

The rails span slightly beyond the 8×8 playing area so the carriage can reach capture zones off to the side of the board. This lets us slide captured pieces completely off the main grid and keeps the motion path clean and collision-free.
