Project Overview

RUBLING is a robotic Rubik’s Cube mural system built to physically produce predefined visual patterns through coordinated mechanical motion. Rather than solving arbitrary cube states, the system focuses on executing carefully planned sequences with consistency and reliability.

A Raspberry Pi coordinates high level control and user interaction, while multiple microcontrollers handle precise motor execution. This division allows the system to remain predictable during demonstrations while clearly exposing the mechanical and software decisions behind its design.
