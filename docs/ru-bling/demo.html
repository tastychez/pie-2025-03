Demo Day

On Demo Day, Wednesday, December 17, 2025, our project finally came to fruition.

What worked?

  • Cube solver - it was able to obtain the correct configuration with 100% accuracy
  • User input - all touch srceen buttons responded and worked as expected to allow the user to follow instrcutions and interact with the solver
  • Almost all of it! The project was able to fulfill its main goal of having a solver that solves a cube to a target pattern using user input.

What didn't work as well?

  • Cube motor detachment system - the force generated by the secondary motors and pulley was simply not strong enough to push the motor with the cube inserts fully into the cube. To get around this, we manually moved the motors into place and used a rubber band to hold the cube motors in place together while the solver ran.
  • Rack-and-Pinion - we had aimed to have a rack-and-pinion pusher to push the solved cube off of the solver once it was done. However, this did not end up working out due to lack of time for testing
