SCRIBBLZ 3D MODEL FILES
========================

Place your OnShape exports here. Export as GLTF (.glb) format.

Required files:
---------------
1. full-assembly.glb          - Main robot assembly (assembled view)
2. full-assembly-exploded.glb - Main robot assembly (exploded view)
3. chassis.glb                - Chassis only
4. centrifugal-impeller.glb   - Impeller model
5. kiwi-drive.glb             - Kiwi drive wheel assembly
6. docking-station.glb        - Docking station (assembled)
7. docking-station-exploded.glb - Docking station (exploded)

How to export from OnShape:
---------------------------
1. Open your document in OnShape
2. Right-click the Part Studio or Assembly tab
3. Click "Export..."
4. Select format: GLTF
5. Click Export and save the .glb file
6. Rename to match the filenames above
7. Place in this folder

For exploded views:
-------------------
1. Create an Exploded View in OnShape (Insert > Exploded View)
2. Position parts as desired
3. Export while in that view
4. Name with "-exploded" suffix
